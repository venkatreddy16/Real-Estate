# Project Guide

## Included features
- Mobile-first responsive layout
- Breakpoints aligned to mobile, tablet, desktop, and large desktop needs
- Theme persistence using localStorage and system preference detection
- RTL support using `dir` switching and supplemental stylesheet rules
- Client-side form validation for auth and search forms
- Skeleton loading effect for dashboard metrics and listing cards
- Shared navigation with dropdown menus and hamburger menu on tablet/mobile

## Important files
- `assets/css/style.css`
- `assets/css/dark-mode.css`
- `assets/css/rtl.css`
- `assets/js/layout.js`
- `assets/js/main.js`
- `assets/js/dashboard.js`

## Notes
- Image assets are stored in `assets/images` across JPG, PNG, AVIF, and video formats
- `robots.txt` and `sitemap.xml` are configured for the LuxEstate production domain and should be updated again if the deployment domain changes
