# LuxEstate Template

Luxury and niche-market real estate HTML template with:
- Home-1 and Home-2 variants
- Public marketing pages
- Single-page login and signup flow
- Admin and user dashboards
- Dark/light mode toggle
- RTL layout toggle
- Local image assets for property, team, and marketing sections
- Shared responsive navigation and footer
- Starter SEO files aligned to the LuxEstate production domain
